Tema 3

Pentru a crea liste se pot folosi tag-urile:
  1. ul - lista neordonata
  2. ol - lista ordonata
  3. li - item din lista
  4. dl - lista cu descrieri
  5. dt - termin dintr-o lista cu descrieri
  6. dd - descrierea dintr-o lista cu descrieri
Pentru a insera un document HTML în altul se poate folosi tag-ul <iframe>.
Astfel, exemplul de mai jos inserează conţinutul de la adresa https://html.spec.whatwg.org/multipage/.
